# Welcome to Markdown Editor

A simple markdown editor built with React and TypeScript.

## Features
- Preview
- Upload files
- Download files
- Markdown syntax
- Syntax highlighting
- File upload/download
- Beautiful typography

### Code Example
```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```
